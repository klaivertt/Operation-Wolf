RESUME :
--------

Ce jeu est un Ray-Shooter inspiré du célèbre jeu d'Arcade "Operation Wolf" sorti en 1987. un jeu de tir classique en scrolling latéral. 
Dans ce jeu vous aurez à tirer sur des vagues de soldats vous attaquant sans toucher les otages qui essayent de fuir.
Cependant vos munition sont limitées, mais des chargeurs ainsi que des soins sont lâchés par certains ennemies.


=========================================
           CONTROLES DU JEU
=========================================

TOUCHES DU MENU :
-----------------
Clique Gauche : Interagir avec les boutons du Menu
Échap  : Fermer le jeu.

TOUCHES DU GAME :
---------------------
Clique Gauche : Tirer / Ramasser objets au sol.
Clique Droit : Recharger.


TOUCHES DU GAME OVER :
----------------------
Espace : Redémarrer la partie.
Échap  : Retourner au Menu.
